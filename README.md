## About Baakh

Baakh: A comprehensive web portal dedicated to preserving and promoting Sindhi poetry. Features include multi-lingual support, auto transliteration, and a rich history section. Bakkh celebrating Sindhi poetry heritage and fostering a global community of poets.

This project is aimed to work on small tools of Baakh and thus it is named *Baakh Data*

## How to use
- Clone this repository or create a pull request
- Grab .sql file from /sql folder and import into the `laravel_baakh` named database
- upgrade / load composer files with command `php artisan composer`
- finally run artisan command `php artisan serve` to start server
